■　作り方等の情報
ATTiny85を使って58曲が再生できるメロディIC「M8058S」を制御するためのソースコードです。
10MΩ可変抵抗で58曲から選曲、選曲している曲番号は7セグメントLEDで表示します。
再生ボタンを押すと再生スタート、停止ボタンで再生停止します。
回路図、作り方含めてブログにまとめていますのでこちらを御覧ください。
